# Convertisseur Markdown vers PDF

Application React pour convertir des fichiers Markdown en PDF.

## Installation

```bash
npm install
```

## Utilisation

```bash
npm start
```

L'application sera disponible sur http://localhost:3000

## Fonctionnalités

- Éditeur Markdown avec aperçu en temps réel
- Importation de fichiers .md/.markdown/.txt
- Conversion en PDF avec mise en forme préservée
- Support des tableaux, code, liens et autres éléments Markdown
- Interface simple et intuitive

## Dépendances principales

- React 18
- react-markdown (rendu Markdown)
- jsPDF (génération PDF)
- html2canvas (capture d'écran HTML)
- remark-gfm (GitHub Flavored Markdown)
- github-markdown-css (style GitHub)
